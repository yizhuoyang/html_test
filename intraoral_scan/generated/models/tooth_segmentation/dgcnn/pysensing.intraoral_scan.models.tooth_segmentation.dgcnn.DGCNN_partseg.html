DGCNN_partseg¶

class pysensing.intraoral_scan.models.tooth_segmentation.dgcnn.DGCNN_partseg(k=25, seg_num_all=33, num_features=8, emb_dims=1024, dropout=0.5)[source]¶

Bases: Module

Implementation of “Dynamic Graph CNN for Learning on Point Clouds” for part segmentation on point clouds.

Parameters:

  • k (int) – Number of nearest neighbors for graph feature extraction (default: 25).

  • seg_num_all (int) – Total number of segmentation classes.

  • num_features (int) – The number of input features for each point.

  • emb_dims (int) – Dimension of the output embedding (default: 1024).

  • dropout (float) – Dropout probability for fully connected layers (default: 0.5).

Note

This model uses a DGCNN-based encoder followed by fully connected layers to perform part segmentation.

forward(x, category)[source]¶

Forward pass for DGCNN-based part segmentation.

Parameters:

  • x (Tensor) – Input point cloud of shape (batch_size, num_points, num_features).

  • category (Tensor) – Category labels of shape (batch_size, 2).

Returns:

Predicted segmentation labels’ score for each point (batch_size, seg_num_all, num_points).

Example

>>> model = DGCNN_partseg()
>>> x = torch.randn(4, 1024, 8)  # batch_size=4, npoints=1024, nfeatures=8
>>> category = torch.randn(4, 2)  # batch_size=4, num_classes=2
>>> output = model(x, category)
>>> print(output.shape)  # Output: (4, 33, 1024)
