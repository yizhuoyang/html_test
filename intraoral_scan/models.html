intraoral_scan.Models¶

Tooth Alignment CurveNet¶

CurveNet

Implementation of "Walk in the Cloud: Learning Curves for Point Clouds Shape Analysis".

Tooth Alignment DGCNN¶

knn

Computes the k-nearest neighbors (KNN) for each point in the point cloud.

get_graph_feature

Constructs the graph feature for each point by concatenating the feature differences between each point and its k-nearest neighbors.

Encoder

Encoder module that extracts hierarchical features from point clouds using a dynamic graph-based approach.

DGCNN

Implementation of "Dynamic Graph CNN for Learning on Point Clouds".

Tooth Alignment TANet¶

custom_max

Performs max pooling along one or multiple dimensions.

PointNet

A simple PointNet encoder that applies MLP layers to point cloud data and performs global max pooling.

GRUCell

Custom GRU cell implementation.

TANet

Implementation of "TANet: Towards Fully Automatic Tooth Arrangement".

Tooth Segmentation DBGANet Utils¶

pc_normalize

Normalize the point cloud so that it is centered at the origin and scaled within unit length.

square_distance

Calculate Euclid distance between each two points.

index_points

Index points using the provided indices.

farthest_point_sample

Perform farthest point sampling on the input point cloud.

query_ball_point

Find the points within a specified radius for each query point.

PointNetSetAbstractionMsg

PointNet Set Abstraction with Multi-Scale Grouping.

centroid

The centroid module, which performs point cloud feature extraction.
