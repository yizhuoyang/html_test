DBGANet¶

class pysensing.intraoral_scan.models.tooth_segmentation.dbganet.DBGANet(num_classes=33, num_neighbor=32)[source]¶

Bases: Module

Implementation of “DBGANet: Dual-Branch Geometric Attention Network for Accurate 3D Tooth Segmentation”

Parameters:

  • num_classes (int) – Number of classes for segmentation. Default is 33.

  • num_neighbor (int) – Number of neighbors to consider for each point in the graph. Default is 32.

forward(x, category)[source]¶

Forward pass for DBGANet, extracting features from input point clouds and performing segmentation.

Parameters:

  • x (Tensor) – Input point cloud data of shape (B, N, 6), where B is the batch size, N is the number of points, and 6 includes coordinates and normals.

  • category (Tensor) – Category input, typically representing specific jaw regions or types.

Returns:

The segmentation result of shape (B, num_classes, N).

Example

>>> x = torch.rand(4, 10000, 6)  # Batch of 4 point clouds, each with 10000 points and 6 features (coords + normals)
>>> category = torch.rand(4, 2)  # Batch of 4, with 2 categories each (e.g., upper or lower jaw)
>>> model = DBGANet(num_classes=33, num_neighbor=32)
>>> output = model(x, category)
>>> print(output.shape) # Expected output: (4, 33, 10000) for segmentation classes of each point
