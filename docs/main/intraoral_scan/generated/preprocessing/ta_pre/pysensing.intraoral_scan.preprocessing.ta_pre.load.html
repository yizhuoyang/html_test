load¶

pysensing.intraoral_scan.preprocessing.ta_pre.load(coord, axis_o, axis_t)[source]¶

Apply transformations to the input coordinates using the original and target axes.

Parameters:

  • coord (Tensor) – The input coordinates of shape (28, 1024, 3), representing the 3D positions.

  • axis_o (Tensor) – The original axis tensor of shape (28, 7), where the first 4 elements are quaternion values and the last 3 elements are translations.

  • axis_t (Tensor) – The target axis tensor of shape (28, 7), structured similarly to axis_o.

Returns:

The transformed coordinates after applying the axis transformations. The combined quaternion and positional transformations after applying both original and target axes.
