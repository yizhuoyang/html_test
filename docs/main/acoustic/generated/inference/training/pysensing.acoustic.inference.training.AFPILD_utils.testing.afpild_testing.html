afpild_testing¶

pysensing.acoustic.inference.training.AFPILD_utils.testing.afpild_testing(config_file='.pysensing/acoustic/inference/training/AFPILD_utlis/configs/afpild_spec_gcc_fusion.json', root_dir='./', dataset_dir='./data/audio_feature', resume_path='saved/AFPILD-CRNN/accil_ori_cloth/model/model_best.pth', device=None, mode='test', run_id='Fusion Model', log_name=None, task='accil_ori_cloth')[source]¶

Function to run the test of the model.

This function executes the testing phase for a machine learning model for AFPILD

Parameters:

  • config_file (str) – Path to the configuration file containing model and test parameters.

  • root_dir (str) – Root directory for model and result saving.

  • dataset_dir (str) – Directory containing the dataset.

  • resume_path (str) – Path to the model checkpoint to resume testing from.

  • device (str) – Device to run the model on (e.g., ‘cpu’, ‘cuda’).

  • mode (str) – Mode of operation. Options: [“train”, “test”]. (Default: “test”)

  • run_id (str) – Unique identifier for the current test run.

  • log_name (str) – Name for the log file to record test results and metrics.

  • task (str) – Specific task or subset of the dataset to run the test on. Slected from: {ide/loc/ideloc/accil}_{ori/ana}_{rd/cloth/shoe}

Returns:

None

Note

This function is based on AFPILD official repo: https://github.com/Cratial/AFPILD-CRNN

Example

>>> from pysensing.acoustic.inference.training.AFPILD_utils.testing import afpild_testing
>>> afpild_testing(
    config_file="./data/AFPILD/afpild_spec_gcc_fusion.json",
    root_dir= "./data/AFPILD",
    dataset_dir="./data/AFPILD/",
    resume_path="./data/AFPILD/saved/AFPILD-CRNN/20241030055348/model/model_best.pth", # Path to the trained model
    task='accil_ori_rd',
)
