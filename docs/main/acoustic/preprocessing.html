Shortcuts

Acoustic.Preprocessing¶

Filtering¶

highpass

Apply biquad highpass filter to obtain the audio above certain frequency

lowpass

Apply biquad lowpass filter to obtain the audio below certain frequency

bandreject

Apply biquad bandrejct filter to reject the audio between certain frequency

bandpass

Apply biquad bandpass filter to obtain the audio between certain frequency

allpass

Apply allpass filter to allow almost all the frequency pass the filter.

Transform¶

stft

Transform the input audio data into its Short-Time Fourier Transform (STFT).

istft

Transform the STFT of a signal back into its time-domain representation using the Inverse Short-Time Fourier Transform (ISTFT).

spectrogram

Convert the input audio data into a spectrogram.

inversespectrogram

Transform the input spectrogram to a raw audio data.

melspectrogram

Transform the input audio data into a MelSpectrogram.

mfcc

Transform the input audio data to a mfcc.

GCC

Transform the input multi-channel audio data to a Generalized Cross-Correlation (GCC) representation.

ipd

Compute Interaural Phase Difference (IPD), cosine of IPD (cosIPD), or sine of IPD (sinIPD) spatial features.

amplitude2db

Transform the spectrogram into dB scale.
