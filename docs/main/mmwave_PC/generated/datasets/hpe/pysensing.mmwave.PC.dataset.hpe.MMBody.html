MMBody¶

class pysensing.mmwave.PC.dataset.hpe.MMBody(root='../../../data/mmpose/', split='train', modalities=['Depth', 'Radar', 'RGB'], test_scenario='Lab1', normalized=True)[source]¶

Bases: Dataset

Implementation of “mmBody Benchmark: 3D Body Reconstruction Dataset and Analysis for Millimeter Wave Radar”.

Point cloud Mesh/Pose reconstruction dataset collected 4D imaging radar, see https://arberobotics.com/wp-content/uploads/2021/05/4D-Imaging-radar-product-overview.pdf. 2 train scenes [“Lab1”, “Lab2”] and 7 test scenes [“Lab1”, “Lab2”, “Furnished”, “Occlusion”, “Rain”, “Smoke”, “Poor_lighting”] are included. Depth and RGB sensors are implemented without calibration (to be implemented).

Parameters:

  • root (str of Path) – Path to the dataset.

  • split (str) – Split of the dataset. Selected from [“train”, “test”]

  • modalities (list of str) – The selected output modalities.

  • test_scenario (str) – Only applicable for test split dataset, the selected scene (from 7 test scenes) for testing.

  • normalized (bool) – Whether normalized data using the ground truth human pose (torso location). If True,

  • points (subtracting all) –
