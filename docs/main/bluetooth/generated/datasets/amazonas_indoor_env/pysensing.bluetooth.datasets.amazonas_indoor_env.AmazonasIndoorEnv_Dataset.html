AmazonasIndoorEnv_Dataset¶

class pysensing.bluetooth.datasets.amazonas_indoor_env.AmazonasIndoorEnv_Dataset(path_dataset, receiver='ALL', normalize=True, train_frac=0.7, train_seed=0, is_train=True)[source]¶

Bases: Dataset

Torch wrapper of the West Michigan Univeristy Waldo Library BLE RSSI dataset.

Basics of the dataset: Number of Bluetooth Beacons: 15 Location coordinates: 2D (X, Y)

Parameters:

  • path_dataset (String) – The path of the root folder for dataset.

  • receiver (String) – The receiver ID.

  • normalize (Boolean) – Select whether to normalize the data. Default: True.

  • train_frac (Float32) – The percentage of training data. Default: 0.7.

  • train_seed (Int) – The seed for generating training data. Default: 0.

  • is_train (Boolean) – Select whether using training set. Default: True.

Note

The items in ‘receiver’ are case-insensitive but must belong to the following list: [‘ALL’, ‘104D’, ‘10B2’, ‘10CE’, ‘1210’, ‘1211’, ‘1212’, ‘121B’, ‘121D’, ‘1F61’, ‘2055’, ‘20B5’] The ROOM_ID provided in the raw data does not match its description in its original paper: https://www.sciencedirect.com/science/article/pii/S2352340924006590 Therefore, the ROOM_ID is not loaded.
