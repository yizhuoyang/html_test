BaseDistanceRegressor¶

class pysensing.bluetooth.models.localization.triangulation.BaseDistanceRegressor(dim, diff_height=0.0)[source]¶

Bases: object

Base class for the functions that maps the BLE vector into the distances from various BLE Beacons.

Parameters:

  • dim (Int) – Dimension of an BLE vector, also known as the number of BLE Beacons.

  • diff_height (Float) – Height difference between the BLE Beacons and the mobile device.

Note

  • Because the height is non-negligible for 2D localization using the range-based sensors (as which BLE Beacons are used), this library only considers the assumption that all BLE Beacons are installed at a same height and the mobile device only moves in the horizontal plane.

  • The default function is derived from the original path loss model for bluetooth signal propagation, which can be expressed as:

\[\begin{aligned} &R = P_{0} - 10 \alpha log_{10}d, \\ &d = \sqrt{d_{h}^{2} + d_{v}^{2}}, \end{aligned} \]

\(d_{h}\) and \(d_{v}\) are the horizontal distance and vertical distance, respectively. \(d_{h}\) is the one we need to estimate based on a given BLE value. \(P_{0}\) and \(\alpha\) denote the transmitted signal strength and the decay factor, respectively.
