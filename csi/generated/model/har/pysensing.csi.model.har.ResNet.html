ResNet¶

class pysensing.csi.model.har.ResNet(dataset, num_classes, ResBlock, layer_list)[source]¶

Bases: Module

ResNet model for Human Activity Recognition. class Bottleneck : Each bottleneck consists of 3 convolutional layers followed by batch normalization operation and activation functions. And adds resudual connection within the bottleneck. class Block : Each block consists of 2 convolutional layers followed by batch normalization operation and activation functions. And adds resudual connection within the block. self.reshape : Reshape the input size into the size of 3 x 32 x 32. self.fc : It consists of a fully-connected layer.

Parameters:

  • dataset (str) – The dataset used for training the model. It can be ‘NTUFi_HAR’, ‘UT_HAR’, or ‘Widar3’.

  • num_classes (int) – The number of classes in the dataset.

  • ResBlock (class) – The class

  • layer_list (list) – The list of the number of blocks in each layer.

Returns:

A PyTorch model for Human Activity Recognition.

Return type:

nn.Module
